# Machine Learning Framework for Biometric Authentication Using ECG

## Overview
This project presents a Machine Learning framework for biometric authentication using Electrocardiogram (ECG) signals. The system uses ECG patterns to securely identify and authenticate users by comparing the performance of different Machine Learning models.

## Features
- ECG signal preprocessing
- Feature extraction
- Biometric authentication using ECG signals
- Machine Learning model comparison
- Performance evaluation and analysis

## Technologies Used
- Python
- Machine Learning
- Support Vector Machine (SVM)
- Artificial Neural Network (ANN)
- Decision Tree
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Project Workflow
1. Load ECG dataset
2. Preprocess ECG signals
3. Extract important features
4. Train Machine Learning models
5. Compare model performance
6. Authenticate users using ECG patterns

## Results
The implemented models were evaluated using standard Machine Learning performance metrics. The project demonstrates that ECG signals can be effectively used as a secure biometric authentication method.
